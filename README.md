Проект доставки еды на Django с базой данных PostgreSQL.
Для запуска на устройстве требуется Docker, команды прописываются в терминале в папке проекта.  

Если контейнер базы данных уже существует, сначала удалите его:
docker rm -f delivery_db

Далее необходимо поднять сервисы:
docker compose up -d

И запустить скрипт для установки зависимостей, миграций и загрузки тестовых данных:
.\run.bat

После этого сервер будет доступен по адресу: http://127.0.0.1:8000/
